SockAddr class

The SockAddr class is a wrapper of a WinSock sockaddr_in structure, for easier use and representation of a IPv4 and port pair. This object provides a series of methods for changing and retrieving the address and port, as well as in its string representation.

Syntax
class SockAddr;
SockAddr sa( ... );

  • ResetAddr() method
