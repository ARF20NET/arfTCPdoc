TCPServerMulti class

The TCPServerMulti class is a implementation of a multiclient server, with a asyncronous accept loop, and receive loop.

Syntax
class TCPServerMulti;
TCPServerMulti server( ... );
Members
  • TCPServerMulti() constructor
  • ExitThreads() method
  • SendBroadcast() method
  • GetClientCount() method
  • GetClient() method
  • GetClientAddr() method
Remarks

This server crates a thread that continously listen and accept connections, and pushes them in a vector to be recv() by another thread. Is a good practice to stop the thrads with ExitThreads() before destructing the object or closing the application.
