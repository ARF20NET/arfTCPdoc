ARF_DISCONNECT_HANDLER macro

Defines disconnectHandler callback function parameter.

Expands to
std::function<void(SockAddr, arf::Exception)> disconnectHandler
Function definition
SockAddr Parameter

Client address and socket.

arf::Exception Parameter

Error (if any).

Remarks

This function is for both graceful disconnection handing, and for internal thread error handling (like a receive error in the receive thread)..

Requirements
Header: arfTCP.h
Introduced in release 1.4.c.
