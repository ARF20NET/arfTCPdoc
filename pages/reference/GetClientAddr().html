GetClientAddr() method

The GetClient() method gets and returns the IP address and port of the client, in a SockAddr object.

Syntax
arf::SockAddr arf::TCPServerMulti::GetClientAddr(int idx);
Parameters
idx

The client's index, in int format.

Return value

A SockAddr object with the client IP address and port.

Remarks

None.

Requirements
Header: arfTCP.h
Introduced in release 1.2
