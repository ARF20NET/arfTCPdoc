TCPServer class

The TCPServer class is a implementation of a single client server, capable of simple bidireccional I/O.

Syntax
class TCPServer : TCPBase;
TCPServer server( ... );
Members
  • TCPServer() constructor
  • Send() method (inherited)
  • Recv() method (inherited)
  • StartRecvLoop() method (inherited)
