SockAddr() constructor

The SockAddr() function creates a SockAddr object, with the specified IP address and port stored internally.

Syntax
arf::SockAddr(sockaddr_in laddr);
arf::SockAddr(unsigned long iaddr, unsigned short port);
Parameters
laddr

A sockaddr_in (contains IP address and port) structure to create the SockAddr from.

iaddr

A IPv4 address in its unsigned long binary representation.

port

A IP port in unsigned short format.

Return value

The SockAddr object.

Remarks

None.

Requirements
Header: arfTCP.h
Introduced in release 1.0
