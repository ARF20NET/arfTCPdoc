SendBroadcast() method

The SendBroadcast() method sends a buffer to all connected clients.

Syntax
void arf::TCPServerMulti::SendBroadcast(char *buff, size_t size, void(*errorHandler)(int));
Parameters
buff

A pointer to a char buffer, to be send.

size

The size of the buffer.

errorHandler

The user defined error handling function.

Return value

None.

Remarks

None.

Requirements
Header: arfTCP.h
Introduced in release 1.2
