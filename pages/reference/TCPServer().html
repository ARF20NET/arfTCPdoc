TCPServer() constructor

The TCPServer() function creates a TCPServer object, initializes a listen sockes, binds, and listens, blocking the current thread until a client connects. When that happends, the internal socket is valid, or if a error occurs, the user defined error handling function is called, and then return.

Syntax
arf::TCPServer(SockAddr listenSockAddr, void (*errorHandler)(int));
arf::TCPServer(unsigned short listenPort, void(*errorHandler)(int));
Parameters
listenSockAddr

A pair of IPv4 address (listen adapter) and listen port in a SockAddr object to listen at.

listenPort

A IP port in unsigned short format to listen at.

errorHandler

The user defined error handling function.

Return value

The TCPServer object.

Remarks

When using the port only constructor, the socket will listen on all adapters (INADDR_ANY), usually localhost (127.0.0.1), and local address, 192.168.x.x.
