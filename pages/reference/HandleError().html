HandleError() function

The HandleError() function provides binary-wide cleanup of WinSock2 API resources allocalted during execution.

Syntax
void arf::HandleError(int err, ARF_ERROR_HANDLER);
Parameters
err

The API error code.

errorHandler

The user error handler.

Return value

None.

Remarks

If errorHandler is null, a arf::Exception will be thrown, the new way of handling errors.

Requirements
Header: arfTCP.h
Introduced in release 1.4
