APIGetLastError() function

The APIGetLastError() function gets the last occured error in the sockets API.

Syntax
int arf::APIGetLastError();
Parameters

None.

Return value

A int describing a error, by its code. See remarks.

Remarks

This function was introduced in the release 1.3 along with linux support, because the need of having a abstraction layer of the API. In Windows (WinSock2) APIGetLastError() is just a wrapper of WSAGetLastError(). In Linux, it returns errno, automatically set by POSIX sockets when a error occurs.

Note that error codes may differ in Linux and Windows. See WinSock2 error codes and Unix/Linux errno codes.
