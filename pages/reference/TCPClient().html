TCPClient() constructor

The TCPClient() function creates a TCPClient object, initializes a socket, and tries to connect, blocking the current thread until connects suscesfully, of fails. When that happends, the internal socket is valid, or if a error occurs, the user defined error handling function is called, and then return.

Syntax
arf::TCPClient(arf::SockAddr peerSockAddr, void (*errorHandler)(int));
Parameters
peerSockAddr

A pair of IPv4 address (listen adapter) and listen port in a SockAddr object to connect to.

errorHandler

The user defined error handling function.

Return value

The TCPClient object.

Remarks

When conneting under normal circunstances, the call will be almost instant.

Requirements
Header: arfTCP.h
Introduced in release 1.0
