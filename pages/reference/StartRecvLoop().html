StartRecvLoop() method

The StartRecvLoop() method creates a asyncronous thread to continuously receives data from the stream, and calls a user defined function to handle it. If a error occurs, the error handing function is called, and the thread is terminated. The user might terminate the thread by returning false from the callback.

Syntax
void arf::TCPBase::StartRecvLoop(bool (*recvHandler)(char*, size_t), void(*errorHandler)(int));
Parameters
recvHandler

The user defined function to handle received data. It must return bool, and take a char* and a size_t.

errorHandler

The user defined error handling function.

Return value

None.

Remarks

A thread is created and then detached, so before the program is terminated, this thread must be terminated to exit cleanly. The thread runs a infinite loop internally, and blocks until data is received, so to terminate the thread the recv call has to be canceled someway.
