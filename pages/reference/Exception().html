Exception() constructor

The Exception() constructor creates a Exception object, and initializes e and APIErrorCode.

Syntax
arf::Exception::Exception();
arf::Exception::Exception(types e, int err);
Parameters
e

The arfTCP error code.

err

The underlaying API error code.

Return value

Exception object.

Remarks

None.

Requirements
Header: arfTCP.h
Introduced in release 1.4
