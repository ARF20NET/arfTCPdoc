ToString() method

The ToString() method returns the IP address and port, in its string representation (x.x.x.x:x).

Syntax
std::string ToString();
Parameters

None.

Return value

The std::string representing the address and port.

Remarks

The IPv4 address is converted to the standard dotted string representation (x.x.x.x), and the port is converted to host order, and converted to decimal (base 10), appended with a ':' separator.

Requirements
Header: arfTCP.h
Introduced in release 1.0
