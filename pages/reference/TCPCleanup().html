TCPCleanup() function

The TCPCleanup() function provides binary-wide cleanup of WinSock2 API resources allocalted during execution.

Syntax
void arf::TCPCleanup();
Parameters

None.

Return value

None.

Remarks

Only required in WinSock2 (Windows). This call won't fail.

Requirements
Header: arfTCP.h
Introduced in release 1.4
