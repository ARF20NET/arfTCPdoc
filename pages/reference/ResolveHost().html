ResolveHost() function

The ResolveHost() function resolves a host, or a IP address onto a unsigned long address for use, via DNS request.

Syntax
unsigned long arf::ResolveHost(const char *name, void(*errorHandler)(int));
unsigned long arf::ResolveHost(std::string& name, void(*errorHandler)(int));
Parameters
name

A null-terminated string (a.k.a. cstring) or a std::string containing a hostname, or a IPv4 address in its text decimal dotted representation (e.g. 62.43.97.153).

errorHandler

A pointer to a user defined error handing function which takes a int as parameter. If NULL, the error will be ignored.

Return value

The host address in unsigned long format. If error occurs, (unsigned long)0 is returned.

Remarks

This function will use the system primary DNS server to resolve the hostname.

The call may fail if the DNS server times out, or the hostname doesn't exist.

Requirements
Header: arfTCP.h
Introduced in release 1.0
