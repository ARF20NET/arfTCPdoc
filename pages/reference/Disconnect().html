Disconnect() method

The Disconnect() method shutdowns send and recv, and then closes the connection of the socket, the socket is invalid after the call, and must be destroyed or reconstructed.

Syntax
void arf::TCPBase::Disconnect(ARF_ERROR_HANDLER);
Parameters
errorHandler

The user defined error handling function.

Return value

None.

Remarks

This call practically will not fail.

Requirements
Header: arfTCP.h
Introduced in release 1.4
