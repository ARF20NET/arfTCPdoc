ToString() method

The ToString() method returns a string, describing the arfTCP error.

Syntax
std::string arf::Exception::ToString()
Parameters

None.

Return value

The std::string message.

Remarks

Possible messages, that follows the e member scheme:

No error.
Successful TCPInit() not yet performed.
Network is down.
Network is unreachable.
Network dropped connection on reset.
Alredy done.
Unknown internal arfTCP error.
Unknown internal unrecoverable arfTCP error.
No more socket descriptors are available.
Invalid address specified.
Hostname not found, or no A entry found.
Software caused connection abort.
Connection reset by peer.
Connection refused by peer.
Connection timed out.

Stored as a private std::vector of std::string.

In 1.4c NOERROROK = 0, was added, making it not backwards compatible.

Requirements
Header: arfTCP.h
Introduced in release 1.4
