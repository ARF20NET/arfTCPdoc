ARF_ACCEPT_HANDLER macro

Defines acceptHandler callback function parameter.

Expands to
std::function<bool(TCPBase*, SockAddr)> acceptHandler
Function definition
bool Return value

Return false to stop accepting clients, otherwise true.

TCPBase* Parameter

Direct I/O to the client. For example to send a welcome message.

SockAddr Parameter

The client IP address and socket.

Remarks

None.

Requirements
Header: arfTCP.h
Introduced in release 1.2.
