TCPServerMulti() constructor

The TCPServerMulti() method creates a TCPServerMulti object, initializes the socket, creates a thread to listen it, and another to recv() the clients.

Syntax
arf::TCPServerMulti(SockAddr listenSockAddr, ARF_ACCEPT_HANDLER, ARF_RECV_HANDLER, ARF_DISCONNECT_HANDLER, ARF_ERROR_HANDLER = nullptr) noexcept(false);
Parameters
listenSockAddr

A pair of IPv4 address (listen adapter) and listen port in a SockAddr object to listen at.

acceptHandler

The user defined function to handle accepted clients. See ARF_ACCEPT_HANDLER.

recvHandler

The user defined function to handle received data. See ARF_RECV_HANDLER.

disconnectHandler

The user defined disconnect and internal thread error handling function. See ARF_DISCONNECT_HANDLER

errorHandler

The user defined error handling function. See ARF_ERROR_HANDLER.

Return value

The TCPServerMulti object.

Remarks

If a error occurs during the listen socket initialization, a exception will be raised. If a error occurs on a underlaying thread, the disconnect handler will be called with error information, without raising a expection. When any error occurs receiving from a client, that client's connection is terminated, and the client is deleted from the client vector.

Requirements
Header: arfTCP.h
