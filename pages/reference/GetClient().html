GetClient() method

The GetClient() method gets and returns a reference to client's TCPBase object with the socket, and I/O methods.

Syntax
arf::TCPBase* arf::TCPServerMulti::GetClient(int idx);
Parameters
idx

The client's index, in int format.

Return value

A pointer to a TCPBase object, with the valid socket of a client.

Remarks

None.

Requirements
Header: arfTCP.h
Introduced in release 1.2
