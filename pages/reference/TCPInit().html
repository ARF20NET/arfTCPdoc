TCPInit() function

The TCPInit() function provides binary-wide initialization of WinSock2 API.

Syntax
void arf::TCPInit();
Parameters

None.

Return value

None.

Remarks

Only required in WinSock2 (Windows). This call won't fail, but theorically it might if something in the OS is broken.

Requirements
Header: arfTCP.h
Introduced in release 1.0
